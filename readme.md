### Project Permohonan Surat pengantar RT / RW


Aplikasi mobile untuk permohonan surat pengantar pada rt rw
```dibuat menggunakan android studio 4.1.1```
beberapa surat yang bisa di request

1. Membuat KTP
2. Membuat Kartu keluarga
3. SKTM
4. Keterangan Miskin
